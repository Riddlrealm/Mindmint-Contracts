# Contributor onboarding

A 30-minute checklist for new contributors:

1. Install Rust + WASM target + Soroban CLI (`scripts/setup-dev.sh` helps).
2. Clone, build (`cargo build --target wasm32-unknown-unknown --release --package <crate>`).
3. Run tests (`scripts/run-tests.sh`).
4. Read `CONTRIBUTING.md`, `docs/CONTRIBUTING_GUIDE.md`, and `CODE_OF_CONDUCT.md`.
5. Find a `good first issue` and tackle it.
