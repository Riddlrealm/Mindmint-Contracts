# ADR-0003: Rename project to Mindmint

## Status

Accepted (2026-07-03).

## Context

The previous working name no longer fit our direction.

## Decision

Rename to Mindmint across `README.md`, `Cargo.toml`, docs, and git history.

## Consequences

- Cleaner brand for downstream apps.
- Existing deploy IDs are unaffected (only metadata changed).
