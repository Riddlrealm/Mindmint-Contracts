# Privacy impact assessment

Mindmint contracts store on-chain data that is publicly readable.

## Data categories

- Public — puzzle IDs, achievement metadata, leaderboard scores.
- Pseudonymous — Soroban addresses used as identities.
- Off-chain recommended — PII should not be stored on-chain.

## Controls

- Encourage users to use fresh identities for high-privacy use cases.
- Document what each contract stores.
