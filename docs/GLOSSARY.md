# Glossary

| Term | Meaning |
|---|---|
| **Soroban** | The smart-contract runtime used by Stellar. |
| **WASM** | The compilation target for Soroban contracts. |
| **Crate** | A Rust package. The workspace has ~110 crates. |
| **Contract ID** | The on-chain address of a deployed contract. |
| **Admin** | The privileged address authorised to call admin-only methods. |
| **Pause** | A global flag that blocks state-mutating operations. |
| **Cross-contract call** | Invocation of one contract from another. |
| **Optimised WASM** | The stripped-down, smaller `.optimized.wasm` produced by the Soroban CLI. |
| **Source identity** | The Stellar account whose secret key signs deploy/invoke transactions. |
