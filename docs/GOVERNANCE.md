# Governance

How decisions are made in this project.

## Roles

- **Maintainers** — merge rights, release authority.
- **Reviewers** — domain experts, advisory votes.
- **Contributors** — open PRs, file issues.

## Decision types

- **Routine** — a maintainer decides.
- **Material** — two-maintainer approval.
- **Breaking** — RFC required, two-week comment window.

See `docs/RELEASE_NOTES_GUIDE.md` for how material changes ship.
