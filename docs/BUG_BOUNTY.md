# Bug bounty program

Mindmint maintains a friendly bug-bounty program for high-severity findings.

## Scope

- All contracts under `contracts/` that are deployed to mainnet.
- Build pipelines under `.github/`.

## Out of scope

- Test/development contracts.
- Sandbox environments.

## Rewards

Discussed case-by-case; severity (SEV1/SEV2/SEV3) drives the tier. Please contact us first.
