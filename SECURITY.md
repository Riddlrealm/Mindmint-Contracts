# Security Policy

If you discover a security vulnerability in Mindmint, please report it privately so we can patch it before disclosure.

## Reporting a Vulnerability

- Email: security@mindmint.example (placeholder — replace with the real address before publishing).
- Use GitHub's "Report a vulnerability" button on the Advisories tab if enabled.

Please do **not** open a public issue for security bugs.

## Response Targets

- We aim to acknowledge new reports within 3 business days.
- Critical issues will be addressed in a hotfix release as soon as possible.
- We follow responsible disclosure: please give us a reasonable window before publishing details.
